# 한국 SV 커뮤니티 어휘 사전

> AI 응답·UI 문구·문서 본문에서 영어 직역 대신 이 표를 따른다.
> 한국 포켓몬 커뮤니티(포붐갤·디시 SV갤·포코카페 등)에서 실제 통용되는 표현 위주.

## 메타·세력 분석

| 영어                | 직역 (금지)         | 권장 한국어 어휘 |
|-------------------|----------------|------------|
| Major threat      | 메이저 위협        | 탑티어 견제픽, 주류 카운터, 상위 카운터 |
| Tier list         | 티어 리스트        | 티어, 픽률 |
| Usage rate        | 사용률            | 사용률 (그대로), 픽률 |
| Counter           | 카운터            | 카운터 (그대로), 견제 |
| Check             | 체크              | 견제 |
| Meta              | 메타              | 메타 (그대로) |
| Niche             | 니치              | 비주류, 마이너 픽 |

## 계산 지표 용어

| 용어        | 의미 |
|-----------|---|
| 결정력       | 공격 측이 방어 측을 얼마나 깎을 수 있는지의 지표 |
| 실질 내구력   | HP·방어·특방·도구·상태를 반영한 실전 내구 추정치 |

## 사용 지침

- 한국 커뮤니티에서 같은 의미로 둘 이상이 쓰이면 더 짧은 쪽 우선 (예: "교체" > "스위치 아웃").
- 사용자(박기린)가 직접 쓴 표현이 있으면 그 표현 우선 채택.
- 모르겠으면 한국 SV 커뮤니티(포붐갤·디시 SV갤·포코카페)에서 통용되는 표현을 따른다.
- 이 표에 없는데 자주 등장하는 단어가 생기면 항목으로 추가한다.
